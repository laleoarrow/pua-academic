# 统计方法决策树

```
开始
  │
  ├── 研究目的是什么？
  │   ├── 描述 → 描述性统计
  │   ├── 比较 → 继续分支
  │   ├── 关联 → 相关/回归
  │   └── 预测 → 机器学习
  │
  ├── 如果是比较：
  │   ├── 组数：2组 → t检验/Mann-Whitney
  │   ├── 组数：>2组 → ANOVA/Kruskal-Wallis
  │   ├── 是否重复测量？
  │   │   ├── 是 → 配对检验/重复测量ANOVA
  │   │   └── 否 → 独立样本检验
  │   └── 数据正态吗？
  │       ├── 是 → 参数检验
  │       └── 否 → 非参数检验
  │
  └── 记得报告：效应量 + 置信区间 + p值
```

### 选择检查清单

| 检查项 | 问自己 |
|-------|--------|
| 数据类型 | 连续？分类？计数？生存时间？ |
| 分布形态 | 正态？偏态？零膨胀？ |
| 样本量 | 大样本可以参数，小样本非参数或精确检验 |
| 独立性 | 独立样本？配对？聚类/重复测量？ |
| 多重比较 | 几个检验？Bonferroni / BH / Holm？ |
| 效应量 | Cohen's d / η² / OR / HR / r？ |
