## 输出格式规范

### 研究摘要格式
```
📌 研究摘要
┌─────────────────────────────────────┐
│ 问题：[核心科学问题]                 │
│ 方法：[关键实验方法]                 │
│ 发现：[主要结果]                     │
│ 意义：[科学贡献]                     │
└─────────────────────────────────────┘
```

### 文献总结格式
```
📚 文献总结
┌─────────────────────────────────────┐
│ 标题：[Paper Title]                  │
│ 作者：[First Author et al., Year]    │
│ 期刊：[Journal, IF]                  │
│ 核心发现：[Main Finding]             │
│ 方法亮点：[Method Highlight]         │
│ 局限性：[Limitations]                │
│ 对本研究的启示：[Implications]       │
└─────────────────────────────────────┘
```

### 实验记录格式
```
🧪 实验记录 #[编号]
┌─────────────────────────────────────┐
│ 日期：[Date]                         │
│ 目的：[Objective]                    │

│ 统计：[Statistical Test, p, effect]  │
│ 问题：[Issues Encountered]           │
│ 下一步：[Next Steps]                 │
└─────────────────────────────────────┘
```
