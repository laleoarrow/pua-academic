## 问卷数据分析框架

### 核心指标计算方法

| 指标 | 计算方式 | PMF 阈值 |
|------|---------|---------|
| **Sean Ellis Score** | Q11选A的人数 ÷ (Q10选A/B/C/D的人数) | ≥40% |
| **方法论vs情绪比** | Q12(a+b+c+d+e) ÷ Q12(f+g) | >1 = 方法论驱动, <1 = 情绪驱动 |
| **留存健康度** | Q10选A+B的比例 | ≥30% |
| **学术痛点验证** | Q5平均分 ≥2.5 的题目数 | ≥4/6 = 需求真实 |
| **传播系数** | Q19(非E)的比例 | ≥40% = 自传播 |
| **不适比** | Q14(非f)的平均选择数 | ≤2 = 可接受 |
| **教练模式需求** | Q16选"教练"或"两者都要"的比例 | ≥30% = 做教练模式 |

### 用户分群

| 群 | 定义 | 关注指标 |
|----|------|---------|
| 核心用户 | Q10=A/B 且 Q11=A | Sean Ellis, Q21 |
| 围观用户 | Q10=D/E | 为什么不持续用？Q14 |
| 方法论用户 | Q12中 a-e 排名最高 | 功能需求，Q17 |
| 情绪用户 | Q12中 f-g 排名最高 | 留存 |
| 场景特定用户 | Q13有明确偏好 | 该场景功能深化 |
| 抗压力用户 | Q14a 被选 且 Q16="教练模式" | 教练模式设计输入 |
